## Default Roles Renamed to Predefined Roles


 PostgreSQL 13 and below used the term “Default Roles”. However, as these roles are not able to actually be changed and are installed as part of the system at initialization time, the more appropriate term to use is “Predefined Roles”. See [Predefined Roles](../../server-administration/database-roles/predefined-roles.md#predefined-roles) for current documentation regarding Predefined Roles, and [the release notes for PostgreSQL 14](../release-notes/prior-releases.md#release-prior) for details on this change.
