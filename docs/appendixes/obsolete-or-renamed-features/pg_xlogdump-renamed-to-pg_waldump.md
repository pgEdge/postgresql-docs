## `pg_xlogdump` renamed to `pg_waldump`


 PostgreSQL 9.6 and below provided a command named `pg_xlogdump`  to read write-ahead-log (WAL) files. This command was renamed to `pg_waldump`, see [pgwaldump](../../reference/postgresql-server-applications/pg_waldump.md#pgwaldump) for documentation of `pg_waldump` and see [the release notes for PostgreSQL 10](../release-notes/prior-releases.md#release-prior) for details on this change.
