## PostGIS Geometry / Geography / Raster Dump Functions


The functions given below are PostGIS functions that take as input or return as output a set of or single [geometry_dump](../postgis-reference/postgis-geometry-geography-box-data-types.md#geometry_dump) or [geomval](../raster-reference/raster-support-data-types.md#geomval) data type object.
