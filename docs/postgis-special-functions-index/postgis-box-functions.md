## PostGIS Box Functions


The functions given below are PostGIS functions that take as input or return as output the box* family of PostGIS spatial types. The box family of types consists of [box2d](../postgis-reference/postgis-geometry-geography-box-data-types.md#box2d_type), and [box3d](../postgis-reference/postgis-geometry-geography-box-data-types.md#box3d_type)
