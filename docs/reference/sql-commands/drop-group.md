# DROP GROUP

remove a database role

## Synopsis


```

DROP GROUP [ IF EXISTS ] NAME [, ...]
```


## Description


 `DROP GROUP` is now an alias for [`DROP ROLE`](drop-role.md#sql-droprole).


## Compatibility


 There is no `DROP GROUP` statement in the SQL standard.
