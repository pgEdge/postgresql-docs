# Version 4.12

Release date: 2019-08-22

This release contains a number of bug fixes and new features since the release of pgAdmin4 4.11.

# New features

[Issue #4144](https://redmine.postgresql.org/issues/4144) -  Add support of Compound Triggers for EPAS 12+.<br>
[Issue #4333](https://redmine.postgresql.org/issues/4333) -  Add support for planner support functions in PostgreSQL 12+ functions.<br>
[Issue #4334](https://redmine.postgresql.org/issues/4334) -  Add support for generated columns in Postgres 12+.<br>
[Issue #4540](https://redmine.postgresql.org/issues/4540) -  Use the full tab space for CodeMirror instances on dialogues where appropriate.<br>
[Issue #4549](https://redmine.postgresql.org/issues/4549) -  Allow a banner to be displayed on the login and other related pages showing custom text.<br>
[Issue #4566](https://redmine.postgresql.org/issues/4566) -  Allow enhanced cookie protection to be disabled for compatibility with dynamically addressed hosting environments.<br>
[Issue #4570](https://redmine.postgresql.org/issues/4570) -  Add an optimisation to the internal code responsible for searching for treeview nodes.<br>
[Issue #4574](https://redmine.postgresql.org/issues/4574) -  Display the row count in the popup message when counting table rows, not just in the properties list.<br>
[Issue #4612](https://redmine.postgresql.org/issues/4612) -  Add support in query history to show internal queries generated by pgAdmin during save data operations.<br>

# Housekeeping

[Issue #4546](https://redmine.postgresql.org/issues/4546) -  Add Reverse Engineered SQL tests for Columns.<br>
[Issue #4554](https://redmine.postgresql.org/issues/4554) -  Add Reverse Engineered SQL tests for Trigger Functions.<br>
[Issue #4555](https://redmine.postgresql.org/issues/4555) -  Add Reverse Engineered SQL tests for Exclusion Constraint.<br>
[Issue #4560](https://redmine.postgresql.org/issues/4560) -  Add a --modules option to the RE-SQL test suite to allow testing of specific object types.<br>
