# Version 6.12

Release date: 2022-07-28

This release contains a number of bug fixes and new features since the release of pgAdmin 4 v6.11.

# New features

[Issue #4488](https://redmine.postgresql.org/issues/4488) -  Added option to trigger autocomplete on key press in the query tool.<br>
[Issue #4607](https://redmine.postgresql.org/issues/4607) -  Allow users to delete files/folders from the storage manager.<br>
[Issue #7389](https://redmine.postgresql.org/issues/7389) -  Allow users to search within the file/storage manager.<br>
[Issue #7486](https://redmine.postgresql.org/issues/7486) -  Added support for visualizing the graphs using Stacked Line, Bar, and Stacked Bar charts in the query tool.<br>
[Issue #7487](https://redmine.postgresql.org/issues/7487) -  Added support for visualise the graph using a Pie chart in the query tool.<br>

# Housekeeping

[Issue #7313](https://redmine.postgresql.org/issues/7313) -  Port the file/storage manager to React.<br>
[Issue #7341](https://redmine.postgresql.org/issues/7341) -  Port change password dialog to React.<br>
[Issue #7342](https://redmine.postgresql.org/issues/7342) -  Port Master Password dialog to React.<br>
[Issue #7492](https://redmine.postgresql.org/issues/7492) -  Removing dynamic module loading and replacing it with static loading.<br>
[Issue #7546](https://redmine.postgresql.org/issues/7546) -  Port named restore point dialog to React.<br>
