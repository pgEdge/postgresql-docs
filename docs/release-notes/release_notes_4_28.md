# Version 4.28

Release date: 2020-11-12

This release contains a number of bug fixes and new features since the release of pgAdmin4 4.27.

# New features

[Issue #3318](https://redmine.postgresql.org/issues/3318) -  Added support to download utility files at the client-side.<br>
[Issue #4230](https://redmine.postgresql.org/issues/4230) -  Added support to rename query tool and debugger tabs title.<br>
[Issue #4231](https://redmine.postgresql.org/issues/4231) -  Added support for dynamic tab size.<br>
[Issue #4232](https://redmine.postgresql.org/issues/4232) -  Added tab title placeholder for Query Tool, View/Edit Data, and Debugger.<br>
[Issue #5891](https://redmine.postgresql.org/issues/5891) -  Added support to compare schemas and databases in schema diff.<br>

# Housekeeping

[Issue #5938](https://redmine.postgresql.org/issues/5938) -  Documentation of Storage Manager.<br>

# Bug fixes

[Issue #4639](https://redmine.postgresql.org/issues/4639) -  Ensure that some fields should be disabled for the trigger in edit mode.<br>
[Issue #5736](https://redmine.postgresql.org/issues/5736) -  Fixed an issue where the validation error message is shown twice.<br>
[Issue #5760](https://redmine.postgresql.org/issues/5760) -  Ensure that non-superuser should be able to debug the function.<br>
[Issue #5842](https://redmine.postgresql.org/issues/5842) -  Ensure that query history should be listed by date/time in descending order.<br>
[Issue #5858](https://redmine.postgresql.org/issues/5858) -  Ensure that search object functionality works with case insensitive string.<br>
[Issue #5895](https://redmine.postgresql.org/issues/5895) -  Fixed an issue where the suffix for Toast table size is not visible in the Statistics tab.<br>
[Issue #5911](https://redmine.postgresql.org/issues/5911) -  Ensure that macros should be run on the older version of Safari and Chrome.<br>
[Issue #5914](https://redmine.postgresql.org/issues/5914) -  Fixed an issue where a mismatch in the value of 'Estimated row' for functions.<br>
[Issue #5919](https://redmine.postgresql.org/issues/5919) -  Added security related enhancements.<br>
[Issue #5923](https://redmine.postgresql.org/issues/5923) -  Fixed an issue where non-closeable tabs are getting closed.<br>
[Issue #5950](https://redmine.postgresql.org/issues/5950) -  Fixed an issue where a long file name is not visible on the process watcher dialog.<br>
[Issue #5953](https://redmine.postgresql.org/issues/5953) -  Fixed an issue where connection to the server is on wait state if a different user is provided.<br>
[Issue #5959](https://redmine.postgresql.org/issues/5959) -  Ensure that Grant Wizard should include foreign tables.<br>
