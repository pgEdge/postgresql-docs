# Getting Started
<a name="getting-started"></a>

pgBackRest strives to be easy to configure and operate:

- [User guides](backrest-page-user-guide-index.md) for various operating systems and PostgreSQL versions.
- [Command reference](backrest-page-command.md) for command-line operations.
- [Configuration reference](backrest-page-configuration.md) for creating pgBackRest configurations.

Documentation for v1 can be found [here](http://www.pgbackrest.org/1). No further releases are planned for v1 because v2 is backward-compatible with v1 options and repositories.
