# SFCGAL Functions Reference

 SFCGAL is a C++ wrapper library around CGAL that provides advanced 2D and 3D spatial functions. For robustness, geometry coordinates have an exact rational number representation.

 Installation instructions for the library can be found on the SFCGAL home page ([http://www.sfcgal.org](http://www.sfcgal.org)). To enable the functions use <code>create extension postgis_sfcgal</code>.
